## collate 

change table collate

```
alter table admins convert to character set utf8mb4 collate utf8mb4_unicode_ci;
```

database

```
ALTER DATABASE laiwang_staging CHARACTER SET = utf8mb4 COLLATE = utf8mb4_unicode_ci;

```
